# Investment Calculator

A simple React.js + TS investment calculating app, a practice project.

# Run the project locally

1. Clone the project

`git clone [link]`


2. Install the dependencies

`npm install`


3. Start the server

`npm run dev`

# Technologies Used 

[Vite](https://vitejs.dev/) for bundling and faster dev environment. 

[TS](https://www.typescriptlang.org/) for strong typization.
